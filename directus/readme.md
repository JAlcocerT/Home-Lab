Spin up a Directus instance via container:

```sh
docker compose -f docker-compose.yml up -d
```
> `http://localhost:8055/admin/content`

Use the DirectusCMS API: https://directus.io/docs/getting-started/use-the-api

Directus and Strapi are two of the most popular open-source, **self-hosted headless CMS platforms**.

They operate with fundamentally different philosophies: a **Database-First** approach (Directus) or a **Code-First** approach (Strapi).
