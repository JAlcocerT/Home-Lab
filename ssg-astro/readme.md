Astro is a **fantastic ssg** that you can combine with:

* StrapiCMS: https://github.com/JAlcocerT/Home-Lab/tree/main/strapi
* Decap-CMS x Astro: See an example with https://github.com/JAlcocerT/Twilight
* Commenting system: https://github.com/JAlcocerT/Home-Lab/tree/main/commento

To create websites completely without code, see [wordpress](https://github.com/JAlcocerT/Home-Lab/tree/main/wordpress) or [ghost](https://github.com/JAlcocerT/Home-Lab/tree/main/ghost).
